# Agent Skills Repository

This repository contains agent skills for AI coding assistants.

## Structure

- `skills/` - Contains all available skills
- Each skill has its own directory with a `SKILL.md` file

## Adding New Skills

1. Create a new directory under `skills/`
2. Add a `SKILL.md` file with proper frontmatter
3. Optionally add `rules/` for categorized guidelines
4. Optionally add `references/` for detailed documentation

## Skill Format

Each skill follows the Agent Skills format with:
- YAML frontmatter (name, description)
- Markdown body with instructions
- Optional supporting files
